Crosswords

CrossWord is a web-based crossword puzzle application that dynamically loads data from a backend represented by a JSON file. This backend can contain information about an unlimited number of crossword puzzles.

All available crosswords are displayed in a user-friendly list on the main page. Users can select any crossword they like, and it will be automatically generated and ready to play. The game allows players to try out answers, check which ones are correct.

To explore a crossword, simply click the "View" button and start solving!

Weather App

WeatherApp is a web application that allows users to enter the name of any city and instantly view the current weather forecast. It fetches real-time weather data by making live API requests to OpenWeatherMap.org.

Simply enter a city, and the app will display key weather details.

To try it out, just click the "View" button and check the forecast for your city!
